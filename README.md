# Ping-Pong
Basic ping-pong on sockets. Written to practice various syscalls.

## Requirements
Listed below are requirements for the code.

### Build system
GCC and CMake

### Syscalls
- `open()` with various flags
- `close()`
- `socket()`
- `listen()`
- `accept()`
- `ioctl()`
- `write()`
- `read()`
